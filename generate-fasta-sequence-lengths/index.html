Generate fasta sequence lengths

2014-08-13   FASTA   gist   

This one liner:

cat file.fa | awk '$0 ~ ">" {print c; c=0;printf substr($0,2,100) "\t"; } $0 !~ ">" {c+=length($0);} END { print c; }
